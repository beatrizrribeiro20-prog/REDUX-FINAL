# Redux Thunk & Redux Persist React Native App

This project was created for IT 320: Mobile Application Development.

## Features

- React Native app built with Expo
- Redux for state management
- Redux Thunk for asynchronous API calls
- Redux Persist for saving state after closing the app
- Loading state using ActivityIndicator
- Error handling state
- Delete user functionality
- Data fetched from Random User API
- ESLint configuration included

## Technologies Used

- React Native
- Expo
- Redux
- React Redux
- Redux Thunk
- Redux Persist
- AsyncStorage

## API Used

Random User API:

https://randomuser.me/api/
